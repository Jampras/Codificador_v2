                 ****CLIQUE UMA VEZ NO BOTAO PARA ALTERAR SUA FUNÇÃO***

Facil de usar e entender 
basta digitar algumas palavras ou texto e clicar no botao "CODIFICAR"
ao lado oque foi digitado mostrara codificado, para ficar mais facil tem um botao de copiar.
basta colar o codigo e apertar o botao " DECODIFICAR "


*caso ja tenha um codigo feito pelo programa*

Nesse caso aperte uma vez no botao para mudar de codificar para Decodificar.
cole o codigo e descubra a mensagem criptografada.
